# 字面量literals

c++高级编程 p245-246

## 1.内置类型的字面量

**整形和浮点型**：

```cpp
int a = 0156;     //8进制，0开头
cout << a << endl;  
a = 0x6A;         //16进制，0x开头
cout << a << endl;
double b = 0xA.123cp-10;  //16进制浮点型，0x开头，cp结尾
cout << b << endl;
a = 0b1110101;    //二进制，0b开头
cout << a << endl;
float c = 3.14f;  //f结尾
cout << c << endl;
```

**字符型**：

```cpp
auto str1 = R"(hello world!)";   //char*
auto str2 = "hello world!"s;    //string
auto str3 = "hello world!"sv;    //string_view
```
